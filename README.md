# 🩺 Smart Health Tracker

Smart Health Tracker is a full-stack MERN application designed to help users monitor and improve their daily health habits through smart tracking, analytics, and wellness insights.

The platform provides an easy and interactive way to track:
- 💧 Water Intake
- 😴 Sleep Hours
- 💪 Exercise Activity
- 😊 Mood Tracking
- 📊 Health Analytics & Charts
